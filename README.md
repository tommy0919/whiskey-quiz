# 🥃 Whiskey Expert Quiz

ウィスキーエキスパート資格試験対策用のインタラクティブなクイズアプリです。

## 📋 出題内容

このアプリには**50問以上**の難問が含まれています。

### カバーするトピック

- **ウィスキーの歴史と地域特性**
  - スコットランド（ハイランド、ローランド、スペイサイド、キャンベルタウン、アイラ）
  - アイルランド
  - 日本
  - アメリカ（バーボン）
  - カナダ

- **製造プロセス**
  - 糖化（モルティング）
  - 発酵
  - 蒸留（ローモンドスチル、ポットスチルなど）
  - 熟成

- **テイスティング知識**
  - ノーズ（香り）
  - パレット（味わい）
  - フィニッシュ（余韻）
  - ボディ（質感）

- **樽と熟成**
  - 樽の種類（バーボン樽、シェリー樽、など）
  - チャリング（焦がし）
  - エンジェルスシェア
  - ダブルマチュレーション

- **ウィスキー分類**
  - シングルモルト
  - ブレンデッドウィスキー
  - グレーンウィスキー
  - ノンエイジステートメント（NAS）

## 🚀 使い方

### ローカルで実行する

1. **Pythonをインストール** （未実装の場合）
   - Python 3.8以上が必要です

2. **必要なパッケージをインストール**
   ```bash
   pip install -r requirements.txt
   ```

3. **アプリを起動**
   ```bash
   streamlit run whiskey_quiz.py
   ```

4. **ブラウザで開く**
   - 自動的に http://localhost:8501 が開きます

### Streamlit Cloudで公開する

1. **GitHubにリポジトリを作成**
   - このファイル（whiskey_quiz.py、requirements.txt、README.md）をGitHubにプッシュ

2. **Streamlit Cloudに登録**
   - https://streamlit.io/cloud にアクセス
   - GitHubアカウントでログイン

3. **アプリをデプロイ**
   - 「Create app」をクリック
   - リポジトリ、ブランチ、ファイル（whiskey_quiz.py）を選択
   - デプロイ完了！

## 📊 クイズの特徴

✅ **ランダム出題** - 毎回異なる順序で出題されます
✅ **リアルタイム採点** - 即座に正誤を判定
✅ **詳しい解説** - 各問題に詳細な説明付き
✅ **スコア追跡** - 進捗状況をリアルタイム表示
✅ **最終評価** - 得点率に応じた評価コメント

## 🎯 推奨される使い方

1. **毎日少しずつ** - 1セッション（50問）を複数回繰り返す
2. **苦手な分野をメモ** - 不正解した問題をリストアップ
3. **解説をしっかり読む** - 正解後の解説を理解する
4. **何度も挑戦** - 90%以上を目指す

## 📈 到達目標

| スコア | 評価 |
|--------|------|
| 90% 以上 | 🏅 素晴らしい！エキスパート水準です |
| 80~89% | ✨ 良好です！あと少しで完璧ですね |
| 70~79% | 📚 もう少し復習が必要です |
| 70% 未満 | 💪 もっと学習しましょう |

## 🔧 カスタマイズ

このアプリは簡単にカスタマイズできます：

- **問題数を増やす** - `quiz_data` リストに新しい問題を追加
- **難易度を調整** - 問題の難度を変更
- **外観を変更** - Streamlitのテーマ設定をカスタマイズ
